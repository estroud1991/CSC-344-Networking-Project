Basic Roommate Stats Updater

1. Run the server first and input the address and port that will be used
	*localhost if you are running on a single computer
	
2. Open the client software and repeate the address and port from step 1

3. Run http server to begin hosting the HTML file created by the server

4. Type in a name and 'returned' or 'departed' then Enter to send to the server

5. Repeat as much as you want

6. Type 'exit' to close when you are done
